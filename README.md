# Project 1
## How to run
* Must have git installed to clone repsitory on terminal.
* Clone the repository
    ```Bash
    git clone https://github.com/3legggedcat/Project1.git
    ```
* Open the file path
    ```Bash
    cd Project1
    ```
* Starting the local webpage
    ```Bash
    start index.html
    ```
