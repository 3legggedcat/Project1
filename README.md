# Project 1
## How to run
* Must have git installed to clone repsitory on terminal.
* Clone the repository
    ```Bash
    git clone <URL for repo>
    ```
* Open the file path
    ```Bash
    cd Project1
    ```
* Starting the local webpage
    ```Bash
    start index.html
    ```
